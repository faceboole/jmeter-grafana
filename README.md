# jmeter-grafana
Jmeter 3.0 + Grafana Latest Official + InfluxDB 0.13.0

##### Build images
```shell
docker-compose build
```

##### Create and start containers:
```shell
docker-compose up
```

##### Links
```
localhost:3000 - grafana (admin/admin)
docker-influxdb (README.md)
```

##### PERFORMANCE TESTS

```mvn verify```

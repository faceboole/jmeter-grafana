# jmeter-grafana
Jmeter 3.0 + Grafana + InfluxDB 

##### Run docker-compose:
```shell
cd docker-compose/
./run.sh
```

##### Create and start compose:
```shell
docker-compose up 
```

##### Start compose:
```shell
docker-compose start 
```

##### Create and start docker container:
```shell
docker-compose up name-of-service
```

##### Start docker container:
```shell
docker-compose start name-of-service
```
